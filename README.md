# Anicode

Find arbitrary unicode characters matching a *search pattern*. The last result match will be copied to your clipboard.

## Install

With [fisherman]

```
fisher anicode
```

## Usage

```
fisher anicode <search pattern>
```

## Example

```
anicode yellow heart
ðŸ’›
```
